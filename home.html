Ce site est un projet des élèves, Yonni Rouelle et Alexandra De Matos, élèves en 3ème année de développement web à l'IMIE de Caen.

Ce projet a pour but le partage de snippet de code pour éviter de chercher dans plusieurs site web différents et de gagner du temps. Il regroupe donc des snippet dans les langages : HTML, CSS, C#, Java, Javascript, Php et Python.
